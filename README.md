# shop-clothing

Shop clothing is a E-commerce app, made for my React learning journey.

The tutorial that is followed uses:
Redux Thunk & Saga,
React Router
Hooks, 
Context API, 
Styled-components,
Payments with Stripe API,
 Firebase to store data,
GraphQL
